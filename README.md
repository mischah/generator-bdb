## Installation

First, install [Yeoman](http://yeoman.io) and generator-bdb using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-bdb
```

Then generate your new project:

```bash
yo bdb
```
